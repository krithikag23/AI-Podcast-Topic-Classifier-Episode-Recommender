# AI Podcast Topic Classifier Episode Recommender
This application uses Natural Language Processing (TF-IDF Vectorization + Logistic Regression)  
to identify the **topic** of a podcast description and recommend similar episodes.

## Features
- **Topic Classification** using ML
- **Top-3 Podcast Recommendations**
- **Prediction Confidence Chart**
- Works in real-time with user input


## Tech Stack
- Streamlit UI
- TF-IDF Vectorizer (Scikit-learn)
- Logistic Regression
- Cosine Similarity for Recommendations
- Plotly Visualizations
